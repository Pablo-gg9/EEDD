## 3. Responda a las preguntas ** sin modificar el codigo**

- ¿Que debería pasar si ingreso 50 euros? : Deberia sumar 50 a la cantidad de saldo anterior de la cuenta
- ¿Que ocurre si ingreso una cantidad negativa? : Nada, el programa seguiria funcionando, pero no tiene mucho sentido si se trata de una Cuenta bancaria.
- ¿Qué pasa si retiro mas dinero del que hay? : Nada, el programa sigue funcionando con una cantidad de saldo actual negativa.
